Detailed Description

The BClock class is a high-precision timer used to calculate delta times between frames.

The BClock class can be used to query the current time since the last update (see getDelta) or since the clock started (see getTime).

[virtual] void BClock::newFrame()

Begins a new frame for the clock.

Calling this function will reset deltaTime to 0 seconds.

[virtual] void BClock::reset()

Resets the current time to 0 seconds.

This function can be called while the clock is running or stopped and can be called multiple times.

[virtual] void BClock::start()

Starts the clock running.

If stop() was previously called without a call to reset(), the clock will continue counting from where it left off.

[virtual] void BClock::stop()

Stops the clock running.

After stop() is called, if start() is called before reset() is called, the clock will continue counting from where it left off.
